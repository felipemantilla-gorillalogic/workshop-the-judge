The Judge

Especificación para Equipos Participantes

Un desafío para practicar habilidades de desarrollo con IA

Desarrolla. Conecta. Compite.

Visión General

The Judge es una plataforma de desafíos de IA donde:

  • Los equipos crean servidores que se conectan a un sistema central
  • El sistema central propone desafíos de procesamiento de texto
  • Los equipos utilizan IA para resolver los desafíos
  • Se otorgan puntos basados en la calidad de las respuestas

Componentes Principales

  • THE JUDGE
    Servidor central que gestiona el juego
  • TEAM SERVER
    Servidor de cada equipo participante
  • OPENAI API
    Utilizada para resolver desafíos

Componentes Necesarios del Team Server

Componentes Esenciales

  • Servidor Web: Para manejar solicitudes HTTP
  • Endpoint de Información: /team-info para registro
  • Cliente HTTP: Para comunicarse con The Judge
  • Cliente OpenAI: Para procesar desafíos
  • Almacenamiento de Estado: Para guardar tokens y desafíos

Tecnologías Recomendadas

  • Node.js + Express: Para el servidor web
  • Axios: Para realizar peticiones HTTP
  • Biblioteca openai: Cliente oficial para OpenAI
  • Variables de entorno: Para configuración
💡

Puedes utilizar cualquier tecnología con la que te sientas cómodo, siempre que cumpla con los requisitos de la API.

Especificación de API - Registro

Registro en The Judge

THE JUDGE
Endpoint:
POST /register
Cuerpo:
{ "callbackURL": "URL de tu endpoint /team-info" }
Respuesta Exitosa:
{ "authToken": "token-auth", "openAiToken": "token-openai" }

Endpoint de Información del Equipo

TEAM SERVER
Endpoint:
GET /team-info (en tu servidor)
Respuesta:
{
  "teamName": "Nombre de tu equipo",
  "members": ["Miembro 1", "Miembro 2", ...],
  "logoURL": "URL a tu logo (opcional)",
  "teamDescription": "Descripción de tu equipo"
}

Especificación de API - Desafíos

Solicitar un Desafío

THE JUDGE
Endpoint:
GET /challenge
Headers:
Authorization: Bearer {authToken}
Respuesta:
{
  "challengeId": "id-único-del-desafío",
  "challenge": "Instrucciones del desafío",
  "input": "Texto a procesar en el desafío"
}

Enviar una Solución

THE JUDGE
Endpoint:
POST /submit
Headers:
Authorization: Bearer {authToken}
Cuerpo:
{ "challengeId": "id-del-desafío", "response": "Tu solución" }
Respuesta:
{
  "score": 85,           // Puntuación (0-100)
  "passed": true,        // true si score >= 70
  "feedback": "Análisis de tu respuesta"
}

Integración con OpenAI

Configuración del Cliente

OPENAI
const OpenAI = require('openai');

// Inicializar cliente con el token recibido
const openai = new OpenAI({
  apiKey: openAiToken  // Token recibido durante el registro
});

Estructura de Prompt Recomendada

OPENAI
const prompt = `
${challenge.challenge}

Texto de entrada:
${challenge.input}

Por favor, responde siguiendo exactamente las instrucciones anteriores.
`;

const completion = await openai.chat.completions.create({
  model: "gpt-3.5-turbo",
  messages: [
    { "role": "system", "content": "Eres un asistente experto en análisis de texto." },
    { "role": "user", "content": prompt }
  ],
  temperature: 0.3,  // Valor bajo para respuestas más precisas
  max_tokens: 150    // Ajustar según necesidad
});

Ejemplo de Desafío Real

Desafío:

Resume el siguiente texto en una sola frase clara.

Input:

La inteligencia artificial está transformando todas las industrias, desde la salud hasta la educación. Los recientes avances en aprendizaje profundo y procesamiento del lenguaje natural han permitido aplicaciones que antes eran imposibles. Sin embargo, también plantea desafíos éticos importantes sobre privacidad, sesgo y el futuro del trabajo humano.

Respuesta ideal:

La inteligencia artificial está revolucionando múltiples sectores con avances tecnológicos significativos, aunque simultáneamente genera importantes desafíos éticos relacionados con la privacidad, el sesgo y el impacto laboral.

Prompt Optimizado

const prompt = `
Resume el siguiente texto en una sola frase clara.

Texto de entrada:
La inteligencia artificial está transformando todas las industrias, desde la salud hasta la educación. Los recientes avances en aprendizaje profundo y procesamiento del lenguaje natural han permitido aplicaciones que antes eran imposibles. Sin embargo, también plantea desafíos éticos importantes sobre privacidad, sesgo y el futuro del trabajo humano.

Tu resumen debe:
1. Capturar tanto los aspectos positivos como los desafíos mencionados
2. Ser conciso pero completo
3. Estar estructurado como una única frase clara
`;

// Temperatura baja (0.2) para respuestas más precisas y consistentes
// max_tokens = 60 es suficiente para una frase concisa
